# Technology Stack

## Recommended technologies

### Backend

- FastAPI: modern Python web framework with automatic docs and strong typing
- SQLAlchemy 2.0: market-standard ORM for scalable persistence
- Pydantic v2: schema validation and typed API contracts
- Uvicorn: performant ASGI server for local development and deployment

## Database

- SQLite: lightweight relational database for the first product phase

## Frontend

- React 18: mature UI library with large ecosystem
- TypeScript: safer contracts between API and UI
- Vite: fast development server and modern bundler
- TanStack Query: reliable server-state management and caching

## Styling and design system

- CSS custom properties: theme tokens driven by monster colors
- Responsive CSS Grid: adaptive catalogue layout
- Component-oriented styling: keeps the card template portable and scalable

## Why this stack fits Southpaw

- professional enough for production-oriented growth
- beginner-friendly compared to heavier enterprise stacks
- easy to host and maintain
- supports rapid iteration on game rules and card content
- gives clean separation between gameplay logic and presentation

## Additional technologies recommended for next phases

- Alembic for database migrations
- Pytest for automated backend tests
- Playwright for frontend end-to-end tests
- Docker for standardised local and deployment environments
- PostgreSQL when multiplayer persistence grows beyond SQLite needs
